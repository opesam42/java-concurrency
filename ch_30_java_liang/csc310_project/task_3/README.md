 # Ayo Game System

A simple Java implementation of the Ayo game (run from the provided package).

## Prerequisites

- Java JDK installed (javac and java available on PATH)

## Compile

From the repository root (the folder named `ch_30_java_liang`), compile all Java files in the `task_3` package:

```bash
javac csc310_project/task_3/*.java
```

## Run

After successful compilation, run the main class:

```bash
java csc310_project.task_3.AyoGameSystem
```

## Notes

- Ensure you run the commands from the repository root so the package paths resolve correctly.
- If you encounter classpath/package errors, verify your current working directory and that the `.class` files were created under the correct package folders.
